Min framtid

I framtiden vill jag kunna programmera webbapplikationer utan några större problem. Mitt mål är att arbeta som frilansare och ha friheten att jobba på distans, men jag kan också tänka mig att arbeta för ett företag. Exakt inom vilket område vet jag inte ännu, men det hade varit roligt att utveckla applikationer som gör nytta för samhället. Jag har även börjat titta på olika traineeprogram som ett möjligt alternativ efter att jag slutar skolan.

Detta vill jag bli bättre på:

Frontendutveckling
Jag vill vässa mina kunskaper i HTML, CSS & JavaScript för att bygga moderna och användarvänliga webbplatser.
React
Jag vill lära mig react för att kunna skapa dynamiska och komponentbaserade webbapplikationer.
Mobilutveckling
Jag vill kunna bygga appar med React Native för både IOS och Android.
